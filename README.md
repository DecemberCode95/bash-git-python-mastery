# Bash, Git y Python Mastery: Plan de 6 Meses

## 🚀 Fase 1: Bash y Terminal Pro (Mes 1)
**Objetivo:** No tocar el mouse. Todo en la terminal integrada de VS Code.

### Ejercicios Diarios (Ciclo Crear-Destruir)
1. **El Laberinto:** Crea una estructura de 5 carpetas anidadas con archivos .txt dentro.
2. **La Purga:** Borra todo lo que no sea .txt usando `rm` y comodines.
3. **El Buscador:** Usa `grep` para encontrar una palabra secreta en 50 archivos generados por un script.

### Mapa Mental: Comandos Vitales
* **Navegación:** `cd`, `ls -la`, `pwd`
* **Manipulación:** `mkdir`, `touch`, `cp`, `mv`, `rm -rf`
* **Inspección:** `cat`, `less`, `head`, `tail`
* **Poder:** `|` (pipes), `grep`, `find`, `chmod`

---

## 🛠 Fase 2: Git y Flujo de Trabajo (Mes 2)
**Objetivo:** Nunca perder una línea de código.

### El Reto de la Destrucción
1. Crea un archivo, haz `git add` y `git commit`.
2. Borra el archivo físicamente (`rm`).
3. **Recupéralo** usando `git checkout` o `git restore`.
4. Arruina un commit intencionalmente y usa `git reset --hard` para volver al pasado.

---

## 🐍 Fase 3: Python para Automatización (Mes 3-4)
**Objetivo:** Que la terminal trabaje por ti.

### Proyectos Sugeridos
* **Script Organigrama:** Un script que tome una carpeta desordenada y mueva archivos a carpetas por extensión (.jpg a /imagenes, etc).
* **Validador de Logs:** Un script que lea un archivo de texto y cuente cuántas veces aparece un error.

---

## 🔐 Fase 4: Ciberseguridad Inicial (Mes 5-6)
**Objetivo:** Aplicar todo lo anterior.

* **OverTheWire (Bandit):** Pasa los niveles 0-20 usando solo lo aprendido.
* **Proyectos:** Un scanner de puertos básico en Python o un script de Bash para automatizar el reconocimiento.
